This information is intended for developers, people who will be writing our code, creating our databases, and designing our interfaces. The mainstream information is located on our Main Page and in the About article.

[edit] This site

Loosely hacked mediawiki, hosted on Dreamhost. Default monobook skin with some hackery to remove the left hand side bar and add the home tab (includes/SkinTemplate.php around line 660 in case you are curious).

Pages of interest that exist and may shed light on the project:

  • About
  • Developers (this page!),
  • Developers/Todo (a somewhat decent strategy doc, actually)
  • Design/GTFSOO
  • Resources/Programming
  • Resources/Programming/GTFS
  • Developers/App ideas
  • if in doubt, consult Special:AllPages.

If in doubt, make a wiki page for it. If you'll be making more edits, please consider registering an account, it should make life easier.

[edit] The code

Basic implementation of Design/GTFSOO in PHP along with (some) code that uses it for manual entry of (some) data is available in our github depot. Contact user:Qviri to be added as a collaborator.

All the code is intended to be open source under a yet-to-be-determined license.

[edit] The data

We'll be, at least for now, using Dreamhost's mysql servers. There is a database created with a roughly appropriate schema, but it is not yet populated by anything meaningful. We plan to make available mysql dumps once there is something more interesting.
